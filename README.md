# ⌨ Typing Trainer

Десктопный клавиатурный тренажёр для обучения слепой печати на русском и английском языках с практикой, уроками, тестами скорости и полноценным игровым режимом.

## ✨ Возможности

### 🎯 Практика
- постепенное открытие букв по заданному порядку для каждой раскладки
- новая буква открывается после нескольких успешных практик
- отображение самой слабой клавиши после завершения
- отдельное уведомление при открытии новой буквы
- общий прогресс открытия букв для практики и игры

### 🧪 Тест скорости
- тесты на `30`, `60` и `120` секунд
- подсчёт скорости, точности, ошибок и общего объёма ввода

### 📖 Уроки
- пошаговое освоение раскладки от простых клавиш к более сложным
- упражнения на отдельные буквы, пары, слова и смешанный текст

### 🕹 Игровой режим
- кампания из `100` уровней
- каждый `5-й` уровень это босс
- `3` жизни на забег
- цель скорости задаётся перед стартом забега
- для боссов действует более строгая точность и отдельный визуальный таймер
- после побед над боссами выдаются награды: открытие буквы или предметы
- между сессиями сохраняется текущий забег, пока игрок не проиграет

### 🧩 Предметы и достижения
- система предметов с редкостью `1–3★`
- обычные реликвии и более рискованные предметы с прочностью
- `3` слота экипировки
- достижения за прогресс, предметы, боссов и полное прохождение игры
- pop-up уведомления о новых достижениях

### 📊 Статистика
- графики прогресса скорости и точности
- история результатов по текущей раскладке
- топ проблемных клавиш
- тепловая карта скорости по символам

### ⌨ Экранная клавиатура
- экранная клавиатура с подсветкой активной клавиши
- визуализация пальцев поверх клавиатуры
- отдельный переключатель `Показывать пальцы` в настройках

## ⌨ Языки и раскладки

Язык и раскладка настраиваются независимо друг от друга.

| Язык | Раскладки |
|------|-----------|
| English | QWERTY, Dvorak |
| Русский | ЙЦУКЕН, ЯВЕРТЫ |

Для русского языка доступен переключатель буквы `Ё`: при отключении все слова, практика и генерация текста работают без неё.

## ⚙ Настройки

- единицы скорости: `WPM`, `CPM`, `CPS`
- стиль и плавность курсора
- глобальный вид текста: блок или бегущая строка
- глобальный размер текста в `rem`
- подсветка текущей буквы
- экранная клавиатура и отдельное отображение пальцев
- язык, раскладка и поддержка `Ё`
- завершение текста пробелом
- сброс всего прогресса или только игрового прогресса

## 🚀 Установка и запуск

### Готовые сборки

Скачайте релиз из раздела [Releases](../../releases):

- **Windows** — `.exe`
- **Linux** — `.AppImage`, `.deb`, `.rpm`

### Запуск из исходников

```bash
git clone https://github.com/DispenserBro/typing-trainer.git
cd typing-trainer
npm install
npm start
```

## 🔧 Сборка

```bash
# Общая сборка проекта
npm run build

# Windows
npm run build:win

# Linux
npm run build:linux
```
