# ⌨ Typing Trainer

Кроссплатформенный клавиатурный тренажёр на Electron с поддержкой русского и английского языков.

## Возможности

| Режим      | Описание |
|------------|----------|
| **Тест**   | Тест скорости печати (30 с / 60 с / 2 мин). Показывает WPM, CPM, точность. |
| **Уроки**  | Поурочное обучение по группам клавиш для каждого пальца (указательный → мизинец). |
| **Практика** | Постепенное открытие букв. Задаётся цель WPM и точности; при достижении — разблокируется следующая буква. Ежедневная цель по минутам или количеству практик. |

### Раскладки
- **QWERTY** (English)
- **Dvorak** (English)
- **ЙЦУКЕН** (Русский)
- **ЯВЕРТЫ** (Русский фонетический)

## Быстрый старт

```bash
# 1. Установить зависимости
npm install

# 2. Запустить приложение
npm start
```

## Сборка

```bash
# Windows (NSIS-инсталлятор)
npm run build:win

# Linux (AppImage)
npm run build:linux
```

## Структура проекта

```
TypingTrainer/
├── main.js              # Electron main process
├── preload.js           # Context bridge
├── package.json
├── data/
│   ├── layouts.json     # Раскладки + уроки + порядок открытия букв
│   ├── words_en.json    # Словарь английских слов
│   └── words_ru.json    # Словарь русских слов
└── renderer/
    ├── index.html
    ├── styles.css
    └── renderer.js      # Вся клиентская логика
```
