# ⌨ Typing Trainer

Десктопный клавиатурный тренажёр для обучения слепой печати на русском и английском языках.

## ✨ Возможности

### 🧪 Тест скорости
Проверьте свою скорость печати за **30 секунд**, **1 минуту** или **2 минуты**. По окончании отображаются скорость (WPM или CPM), точность и количество ошибок.

### 📖 Уроки
Пошаговое обучение от указательных пальцев к мизинцам. Каждый урок содержит **5 упражнений** разного типа: отдельные клавиши, пары, слова, фразы и смешанный текст. Для перехода к следующему упражнению нужна точность ≥ 80%.

### 🎯 Практика
Постепенное открытие букв — начинаете с двух и продвигаетесь дальше, достигая заданной скорости и точности. Есть ежедневная цель по минутам или количеству практик. Отображается самый слабый символ для целенаправленной тренировки.

### 📊 Статистика
Графики прогресса скорости и точности по каждому режиму. История всех сессий.

## ⌨ Раскладки и языки

Язык и раскладка настраиваются **независимо друг от друга** — при выборе языка предлагаются только совместимые раскладки.

| Язык     | Раскладки               |
|----------|-------------------------|
| English  | QWERTY, Dvorak          |
| Русский  | ЙЦУКЕН, ЯВЕРТЫ (фонет.) |

Для русского языка доступен **переключатель буквы Ё** — при отключении все слова, уроки и практика работают без неё.

## ⚙ Настройки

- Единица скорости: **WPM** или **CPM**
- Стиль курсора: подчёркивание, блок, линия
- Плавная анимация курсора
- Экранная клавиатура с подсветкой пальцев
- Звук при ошибке

## 🚀 Установка и запуск

### Готовые сборки

Скачайте установщик из раздела [Releases](../../releases):
- **Windows** — `.exe` (NSIS-инсталлятор)
- **Linux** — `.AppImage`

### Запуск из исходников

```bash
# Клонировать репозиторий
git clone https://github.com/DispenserBro/typing-trainer.git
cd typing-trainer

# Установить зависимости
npm install

# Запустить
npm start
```
